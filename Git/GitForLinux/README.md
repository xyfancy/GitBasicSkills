# Linux下安装Git

---
## 1.使用包管理安装Git


Ubuntu: `sudo apt-get install git`

Centos: `sudo yum install git`

---
## 2.安装zsh对Git支持

### 1.安装zsh

Ubuntu: `sudo apt-get install zsh` 

Centos: `sudo yum install zsh`

### 2.安装oh-my-zsh**

`wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | sh`

###  3.切换Shell

`chsh -s /bin/zsh`

### 4.重启

`sudo reboot`
