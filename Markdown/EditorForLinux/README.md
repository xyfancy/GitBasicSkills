# Linux下安装Markdown编辑器

---
## 1.下载安装Moeditor

https://moeditor.github.io/

## 2.解压后添加路径

解压到Target_Dir中

对于zsh:在~/.zshrc中添加 `export PATH=$HOME/Target_Dir/Moeditor-linux-x64:$PATH`

对于bash:在~/.bashrc中添加 `export PATH=$HOME/Target_Dir/Moeditor-linux-x64:$PATH`


## 3.source 或者重新启动shell

对于zsh:`source ~/.zshrc`

对于bash:`source ~/.zshrc`
